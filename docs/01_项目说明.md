# 项目说明

## 1. 项目定位

`D:\BiliAgent4J` 是 `D:\agent` 的 Java 重构版。

重构原则：

- 保留当前 Java/Maven 工程结构。
- 不删除 Java 代码，不把项目替换成 Python 项目。
- 参考 `D:\agent` 的功能、逻辑、流程、界面和文档。
- 用 Java 持续实现并对齐 Python 项目的最新能力。

## 2. 当前功能范围

### 2.1 内容创作

- 输入领域、方向、想法、分区、风格。
- 输出选题建议、脚本、标题、简介、标签、置顶评论。

### 2.2 视频分析

- 输入 B 站视频链接。
- 自动解析视频信息。
- 输出表现判断、原因分析、优化建议与参考视频。

### 2.3 智能助手

- 在 `LLM Agent` 模式下可用。
- 支持连续对话。
- 支持按需调用工具而不是只做纯文本回答。

### 2.4 知识库管理

- 上传 `.txt/.md/.docx/.pdf` 文件。
- 同步 B 站热门榜样本。
- 查看样本文档。
- 按关键词检索知识内容。

### 2.5 长期记忆

- 记录模块执行结果和助手交互结果。
- 下次任务可自动加载历史经验。

## 3. 当前技术实现

- 后端：`Spring Boot + Java 17`
- LLM：`LangChain4j + OpenAI 兼容接口`
- 数据库：`SQLite`
- 知识库：Java 本地 JSON 向量存储
- 嵌入：Java 本地确定性 embedding
- 文档解析：`Apache POI` / `PDFBox`
- 前端：Spring 静态资源页面

## 4. 与 Python 参考项目的关系

- 功能目标对齐 `D:\agent`。
- 实现语言改为 Java。
- Python 侧依赖的部分能力在 Java 版采用等价方案实现。
- 当前知识库不是复制 Python 文件，而是用 Java 重新实现上传、切片、检索、同步与状态管理。
