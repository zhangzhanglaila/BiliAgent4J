# 项目说明

## 1. 项目定位

`D:\BiliAgent4J` 是 `D:\agent` 的 Java 重构版。

重构原则：

- 保留当前 Java/Maven 工程结构。
- 不删除 Java 代码，不把项目替换成 Python 项目。
- 参考 `D:\agent` 的功能、逻辑、流程、界面和文档。
- 用 Java 持续实现并对齐 Python 项目的最新能力。

## 2. 当前功能范围

### 2.1 内容创作

- 输入领域、方向、想法、分区、风格。
- 输出选题建议、脚本、标题、简介、标签、置顶评论。

### 2.2 视频分析

- 输入 B 站视频链接。
- 自动解析视频信息。
- 输出表现判断、原因分析、优化建议与参考视频。
- **支持 SSE 流式推送**（异步分析模式）。

### 2.3 智能助手

- 在 `LLM Agent` 模式下可用。
- 支持连续对话。
- 支持按需调用工具而不是只做纯文本回答。
- **支持 Chat Session 持久化**（页面刷新后会话不丢失）。

### 2.4 知识库管理

- 上传 `.txt/.md/.docx/.pdf` 文件。
- 同步 B 站热门榜样本。
- 查看样本文档。
- 按关键词检索知识内容。
- **支持 Chroma HTTP API**（远程向量存储）。
- **支持语义嵌入服务**（调用远程 embedding 模型）。

### 2.5 长期记忆

- 记录模块执行结果和助手交互结果。
- 下次任务可自动加载历史经验。

### 2.6 LLM Agent 高级功能

- **工具预算控制**：per-tool 次数限制、最大步数限制、重复调用限制。
- **工具超时处理**：WebSearch、CodeInterpreter 等工具支持超时控制。
- **完整 Reflection**：支持结果质检与自动重写。
- **多候选生成**：支持生成多个候选结果并自评分选择最优。
- **PowerShell HTTP Fallback**：Windows 环境网络问题时自动切换。

## 3. 当前技术实现

- 后端：`Spring Boot + Java 17`
- LLM：`LangChain4j + OpenAI 兼容接口`
- 数据库：`SQLite`
- 知识库：Chroma HTTP API + JSON Fallback
- 嵌入：语义嵌入服务 + 确定性 embedding Fallback
- 文档解析：`Apache POI` / `PDFBox`
- 前端：Spring 静态资源页面
- SSE：`SseEmitter` 流式推送

## 4. 与 Python 参考项目的关系

- 功能目标对齐 `D:\agent`。
- 实现语言改为 Java。
- Python 侧依赖的部分能力在 Java 版采用等价方案实现。
- 当前知识库不是复制 Python 文件，而是用 Java 重新实现上传、切片、检索、同步与状态管理。
- 详细功能对比见 [09_功能对比与同步说明.md](09_功能对比与同步说明.md)。
