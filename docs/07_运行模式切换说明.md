# 运行模式切换说明

## 1. 页面入口

首页顶部包含运行模式区域：

- 模式标题
- 开关按钮
- LLM 配置表单

## 2. 切换规则

### 2.1 开启 LLM Agent 模式

需要满足：

- 已填写 `URL`
- 已填写 `API Key`
- 已填写 `provider`
- 可选填写 `model`

保存成功后，页面会立即切到 `LLM Agent` 模式。

### 2.2 关闭 LLM Agent 模式

- 页面回退到无 Key 逻辑模式
- 不会删除已保存的运行时配置

## 3. 配置来源

优先顺序通常为：

1. 页面运行时填写
2. `.env` 中的默认配置

## 4. 页面状态反馈

切换后会同步更新：

- 顶部模式描述
- Token 消耗提示
- 智能助手可用状态
- 智能助手输入框提示文案
