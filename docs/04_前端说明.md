# 前端说明

## 1. 前端目录

- [index.html](D:/BiliAgent4J/src/main/resources/static/index.html)
- [app.js](D:/BiliAgent4J/src/main/resources/static/app.js)
- [style.css](D:/BiliAgent4J/src/main/resources/static/style.css)

## 2. 当前页面结构

首页已经对齐 `D:\agent` 的三模块结构：

- 视频分析
- 内容创作
- 知识库管理

右侧为智能助手面板。

## 3. 知识库前端实现

知识库模块包含以下子页签：

- 上传知识文件
- 同步热门样本
- 查看知识库
- 关键词检索

关键 DOM 节点：

- `#knowledgePanel`
- `#knowledgeStatusBar`
- `#knowledgeActionHost`
- `#knowledgeStageResult`

## 4. 核心前端函数

- `loadKnowledgeBaseStatus()`
- `uploadKnowledgeFile()`
- `updateKnowledgeBase()`
- `loadKnowledgeSamples()`
- `searchKnowledgeContent()`
- `resumeKnowledgeUpdateJob()`
- `pollKnowledgeUpdateJob()`

## 5. 对接的后端接口

- `GET /api/knowledge/status`
- `GET /api/knowledge/sample`
- `GET /api/knowledge/search`
- `POST /api/knowledge/upload`
- `POST /api/knowledge/update/start`
- `GET /api/knowledge/update/{jobId}`

## 6. 当前前端差异说明

- UI 结构对齐 Python 版。
- 页面运行在 Spring 静态资源模式下，不依赖 Jinja 模板渲染。
- `window.__INITIAL_RUNTIME__` 由静态默认值启动，后续通过 `runtime-info` 拉取真实状态。
