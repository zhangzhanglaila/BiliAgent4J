# 完整部署文档

## 1. 环境要求

- `JDK 17`
- `Maven 3.9+`
- 可访问的 OpenAI 兼容 LLM 接口
- Windows 环境下建议有可写的 Maven 本地仓库目录

## 2. 关键配置

参考 [`.env.example`](D:/BiliAgent4J/.env.example)。

重点变量：

- `LLM_PROVIDER`
- `LLM_API_KEY`
- `LLM_BASE_URL`
- `LLM_MODEL`
- `SERPAPI_API_KEY`
- `VECTOR_DB_PATH`
- `DB_PATH`
- `DEFAULT_PARTITION`
- `DEFAULT_PEER_UPS`

## 3. 启动步骤

### 3.1 开发运行

```bash
mvn spring-boot:run
```

默认端口：

```text
http://127.0.0.1:8000
```

### 3.2 当前机器 Maven 仓库权限异常时

```powershell
$env:MAVEN_OPTS='-Dmaven.repo.local=D:\BiliAgent4J\.m2\repository'
mvn spring-boot:run
```

### 3.3 运行测试

```bash
mvn test
```

### 3.4 打包

```bash
mvn -DskipTests package
```

## 4. 数据目录

- SQLite：由 `DB_PATH` 决定
- 知识库存储：由 `VECTOR_DB_PATH` 决定
- embedding 缓存目录：由 `EMBEDDING_CACHE_DIR` 决定

## 5. 首次验收建议

1. 打开首页，确认三个模块页签都显示正常。
2. 测试视频解析与内容创作接口。
3. 上传一个 `.txt` 或 `.docx` 文件，确认知识库文档数增加。
4. 触发热门知识同步，确认可以看到异步任务进度。
5. 在开启 `LLM Agent` 模式后测试智能助手。
