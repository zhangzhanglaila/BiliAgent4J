# 前端使用手册

## 1. 视频分析

1. 输入 B 站视频链接。
2. 系统先自动解析视频信息。
3. 点击“解析并分析视频”。
4. 查看表现判断、原因分析、优化建议和参考视频。

## 2. 内容创作

1. 输入领域、方向、想法。
2. 选择内容分区和文案风格。
3. 点击“一键生成选题与文案”。
4. 复制标题、脚本、简介、标签和置顶评论。

## 3. 知识库管理

### 3.1 上传知识文件

- 支持 `.txt/.md/.docx/.pdf`
- 上传后会自动提取文本、切片并写入知识库

### 3.2 同步热门样本

- 设置每个榜单抓取条数
- 点击“自动更新热门知识库”
- 页面会显示异步任务进度与写入结果

### 3.3 查看知识库

- 读取当前知识库存储中的样本文档
- 可直接复制正文

### 3.4 关键词检索

- 选择分类或输入关键词
- 查看命中的知识文档与相关度

## 4. 智能助手

- 只有 `LLM Agent` 模式可用
- 助手会自动结合页面上下文
- 助手可以调用知识检索、联网搜索、代码解释器等工具

## 5. 常见问题

### 5.1 为什么助手不可用

- 当前仍在无 Key 逻辑模式
- 没有填写可用的 LLM 配置
- 运行模式开关尚未打开

### 5.2 为什么知识库状态显示本地 JSON

- 当前 Java 版默认使用本地 JSON 向量存储兜底实现
- 这属于 Java 侧的重构方案，不是复制 Python 运行环境
