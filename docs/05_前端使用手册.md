# 前端使用手册

## 1. 启动后先看哪里

打开：

```text
http://127.0.0.1:8000
```

页面顶部会先显示当前运行模式：

- 无 Key 逻辑模式
- LLM Agent 模式

如果当前没有可用的 LLM 配置：

- 打开页面顶部的模式开关时，会先弹出运行时配置表单
- 需要填写 `URL / Key / provider / model`
- 保存后会立即进入 `LLM Agent` 模式

## 2. 内容创作模块怎么用

输入：

- 你的领域
- 你的方向
- 你的想法
- 内容分区
- 文案风格

点击：

```text
一键生成选题与文案
```

你会得到：

- 3 个选题方向
- 推荐主选题
- 标题
- 脚本
- 简介
- 标签
- 置顶评论

## 3. 视频分析模块怎么用

输入：

- 一个 B 站视频链接

页面会先自动做一遍链接解析，显示：

- 标题
- 分区
- UP 主
- BV 号
- 播放、点赞、投币、收藏、评论、分享

然后点击：

```text
一键解析并分析视频
```

你会得到：

- 当前视频真实信息
- 热门爆款 / 播放偏低 判断
- 原因分析
- 优化建议
- 后续可做选题
- 参考视频
- 如果表现偏低，还会额外给一版新文案

## 4. 智能助手怎么用

前提：

- 必须先让页面进入 `LLM Agent` 模式
- 这组配置可以来自 `.env` 默认值，也可以来自页面顶部运行时填写

然后可以直接提问，例如：

- 帮我选题
- 帮我分析这个视频
- 帮我写一版文案

## 5. 常见使用建议

### 做内容创作时

不要只填一个模糊大词，最好至少给出：

- 方向
- 想法

### 做视频分析时

尽量贴完整链接，不要只贴一句“帮我看看这个视频”。

## 6. 复制和参考视频

前端支持：

- 一键复制标题
- 一键复制脚本
- 一键复制完整文案
- 点击参考视频卡片直接打开 B 站页面
