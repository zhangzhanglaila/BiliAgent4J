# 功能对比与同步说明

## 1. 概述

本文档说明 `D:\BiliAgent4J`（Java 版本）与 `D:\agent`（Python 版本）之间的功能对比及同步状态。

## 3. 架构差异说明

### 3.1 向量存储

**Python 版本**：
- 直接使用 `langchain_chroma` 或 `chromadb` Python 库
- 本地持久化到指定目录

**Java 版本**：
- 通过 Chroma HTTP API 与远程 Chroma 服务通信
- 当 Chroma 服务不可用时，自动回退到 JSON 文件存储
- 配置项：`app.chroma-host`、`app.chroma-port`

### 3.2 嵌入模型

**Python 版本**：
- `SentenceTransformer` 直接加载本地模型
- 模型缓存到 `~/.cache/huggingface/hub`

**Java 版本**：
- 通过 HTTP API 调用远程 embedding 服务
- 远程服务部署 Python sentence-transformers 模型
- 当远程服务不可用时，自动回退到本地 deterministic 嵌入
- 配置项：`app.embedding-service-url`

### 3.3 Chat Session

**Python 版本**：
- Redis 缓存 + 本地 JSON 文件持久化
- 支持分布式部署

**Java 版本**：
- 本地 JSON 文件持久化（`vector_db/chat_sessions/`）
- 内存缓存（LRU）
- 适合单实例部署
